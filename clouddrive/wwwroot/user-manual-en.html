2. Getting Started

2.1 System Requirements

CloudDrive Modern runs in any modern web browser. For optimal performance, we recommend:

  • Browser: Chrome 90+, Firefox 88+, Safari 14+, or Edge 90+
  • Internet Connection: Stable broadband connection for cloud operations
  • Screen Resolution: 1024x768 minimum (responsive design adapts to all sizes)
  • JavaScript: Must be enabled

2.2 Installation

CloudDrive Modern is a web application that requires no installation. Simply:

1 Open your web browser and navigate to the CloudDrive Modern URL provided by your administrator.
2 Bookmark the page for easy access in the future.
3 For mobile devices, you can add the web app to your home screen for a native app-like experience.

2.3 First Login

1 Enter your username and password on the login screen.
2 Click the "Login" button or press Enter.
3 If you don't have an account, click "Register" to create a new account.
üí° Tip: Enable "Remember me" to stay logged in across browser sessions. For security, this is disabled on public computers.

3. User Interface

3.1 Navigation

The CloudDrive Modern interface consists of several key areas:

  • Sidebar: Folder tree navigation and quick access to drives
  • Header Bar: View controls, sorting options, and action buttons
  • Main Content: File and folder display area
  • Properties Panel: Detailed file information (can be toggled)
  • Context Menus: Right-click for quick actions

3.2 View Modes

CloudDrive Modern offers three view modes to suit different preferences:

View Mode Description Best For
Grid View Files displayed as cards with thumbnails Visual browsing, image collections
List View Compact table with file details Detailed information, large file lists
Waterfall View Pinterest-style masonry layout Mixed media, visual exploration
üí° Tip: Use the keyboard shortcut V to quickly cycle through view modes.

3.3 Sorting and Filtering

Organize your files efficiently with powerful sorting and filtering options:

Sorting Options

  • Name: Alphabetical order (A-Z or Z-A)
  • Size: File size (smallest to largest or vice versa)
  • Type: File extension/type
  • Modified Date: Last modification time

Advanced Filtering

  • Search by filename or content
  • Filter by file type (images, videos, documents, etc.)
  • Date range filtering
  • Size range filtering

4. File Operations

4.1 Uploading Files

CloudDrive Modern supports multiple upload methods:

Method 1: Drag and Drop

1 Select files or folders from your computer's file explorer.
2 Drag them over the CloudDrive window.
3 Drop them in the desired location (main area or specific folder).

Method 2: Upload Button

1 Click the "Upload" button in the header bar.
2 Select files from the file picker dialog.
3 Click "Open" to start the upload.
üí° Tip: Upload progress is shown in real-time with progress bars and notifications. Large files are uploaded in chunks for reliability.

4.2 Downloading Files

Download files individually or in bulk:

Single File Download

  • Right-click on a file and select "Download"
  • Double-click on non-media files
  • Use the download button in the properties panel

Multiple File Download

1 Select multiple files using Ctrl+Click or Shift+Click.
2 Click the "Download" button in the header bar.
3 Files will be packaged and downloaded as a ZIP archive.

4.3 Moving and Copying

Organize your files by moving or copying them between folders:

Drag and Drop Method

1 Select the files you want to move.
2 Drag them to the destination folder in the sidebar or main area.
3 Hold Ctrl while dropping to copy instead of move.

Context Menu Method

1 Select files and right-click to open the context menu.
2 Choose "Move to..." or "Copy to...".
3 Select the destination folder in the folder picker dialog.
‚ö†Ô∏è Warning: Moving files between different cloud storage providers may take longer as data needs to be transferred. The operation will be performed in the background.

4.4 Renaming and Deleting

Renaming Files and Folders

  • Right-click and select "Rename"
  • Press F2 when a file is selected
  • Click the edit button in the properties panel

Deleting Files and Folders

  • Select items and press Delete
  • Right-click and select "Delete"
  • Use the delete button in the header bar
‚ö†Ô∏è Warning: Deleted files may not be recoverable depending on your cloud storage provider's policies. Always confirm before deleting important files.

5. Folder Management

5.1 Creating Folders

1 Click the "+" button in the header bar or right-click in an empty area.
2 Select "New Folder" from the menu.
3 Enter the folder name and press Enter or click "Create".

5.2 Encrypted Folders

Create encrypted folders for sensitive data:

1 Click the "+" button and select "New Encrypted Folder".
2 Enter a folder name and a strong password.
3 Confirm the password and click "Create".
‚ö†Ô∏è Important: Encrypted folder passwords cannot be recovered. Make sure to remember your password or store it securely.

5.3 Folder Navigation

Navigate efficiently through your folder structure:

  • Sidebar Tree: Click folders in the tree view for quick navigation
  • Breadcrumb Navigation: Click any part of the path to jump to that level
  • Back/Forward: Use browser back/forward buttons or keyboard shortcuts
  • Up Button: Click to go to the parent folder

6. Cloud Storage Integration

6.1 Adding Cloud Storage

Connect your cloud storage accounts to manage all your files in one place:

1 Click the "+" button and select "Add Cloud Storage".
2 Choose your cloud storage provider (Google Drive, OneDrive, etc.).
3 Follow the authentication process to grant access.
4 Your cloud storage will appear in the sidebar.

6.2 Managing Cloud Accounts

  • Refresh: Right-click on a cloud storage and select "Refresh" to sync
  • Disconnect: Remove access to a cloud storage account
  • Quota: View storage usage and available space
  • Settings: Configure sync options and preferences

7.3 Search Functionality

Find files quickly with powerful search features:

Basic Search

1 Click the search icon or press Ctrl+F.
2 Enter your search query.
3 Press Enter to search.

Advanced Search Options

  • Fuzzy Matching: Find files with similar names
  • Content Search: Search within file contents (where supported)
  • File Type Filters: Limit search to specific file types
  • Date Range: Find files modified within a specific time period

7.4 File Sharing

Share files and folders with others:

1 Right-click on a file or folder and select "Share".
2 Configure sharing permissions and expiration.
3 Copy the generated share link.
4 Send the link to recipients.

7.5 Offline Downloads

Queue downloads for offline access:

  • Add files to the offline download queue
  • Downloads continue in the background
  • Access files when internet is unavailable
  • Manage download priorities and bandwidth

9. Troubleshooting

Common Issues and Solutions

Upload Failures

  • Large Files: Check file size limits for your cloud storage
  • Network Issues: Verify internet connection stability
  • Storage Full: Check available storage space
  • File Type: Some cloud providers restrict certain file types

Login Problems

  • Wrong Credentials: Verify username and password
  • Account Locked: Contact your administrator
  • Browser Issues: Clear cache and cookies
  • Network: Check firewall and proxy settings

Performance Issues

  • Slow Loading: Try reducing the number of files displayed per page
  • Memory Usage: Close unused browser tabs
  • Network Speed: Check your internet connection
  • Browser Cache: Clear browser cache and refresh

Cloud Storage Issues

  • Authentication Expired: Re-authenticate your cloud accounts
  • Sync Conflicts: Resolve file conflicts manually
  • API Limits: Wait for rate limits to reset
  • Service Outages: Check cloud provider status pages

10. Frequently Asked Questions

Q: Is my data secure in CloudDrive Modern?

A: Yes, CloudDrive Modern uses industry-standard security practices including HTTPS encryption, secure authentication, and optional file encryption. Your cloud storage credentials are encrypted and stored securely.

Q: Can I access CloudDrive Modern offline?

A: CloudDrive Modern requires an internet connection for most features. However, you can use the offline download feature to cache files locally for offline access.

Q: How much storage space do I have?

A: Storage space depends on your cloud storage accounts and local server configuration. Check the storage information in the sidebar or properties panel.

Q: Can I share files with people who don't have accounts?

A: Yes, you can create share links that allow external users to access specific files or folders without requiring an account.

Q: What happens if I delete a file?

A: Deleted files are removed from CloudDrive Modern and may be moved to your cloud storage provider's trash/recycle bin, depending on the provider's policies.

Q: Can I upload folders?

A: Yes, you can drag and drop entire folders, and CloudDrive Modern will preserve the folder structure during upload.

Q: Is there a mobile app?

A: CloudDrive Modern is a responsive web application that works on mobile devices. You can add it to your device's home screen for a native app-like experience.

Q: How do I report bugs or request features?

A: Contact your system administrator or use the feedback option in the application settings menu.
