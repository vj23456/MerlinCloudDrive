1. 概述

CloudDrive Modern 是一个功能强大的基于网页的文件管理应用程序，为管理多个云存储服务和本地存储中的文件提供统一的界面。采用现代网络技术构建，为组织、访问和共享文件提供响应式、直观的体验。

🌐 多云支持

连接和管理来自多个云存储提供商的文件，包括 Google Drive、OneDrive、Dropbox 等。

📱 响应式设计

针对桌面、平板和移动设备进行优化，具有触摸友好的控件和自适应布局。

🔒 安全功能

加密文件夹、安全身份验证和隐私控制，确保您的数据安全。

🎵 媒体播放

内置媒体查看器，支持图像、视频、音频文件和外部播放器集成。

🔍 高级搜索

强大的搜索功能，支持模糊匹配和内容过滤。

⚡ 高性能

针对处理大型文件集合进行优化，具有智能缓存和延迟加载。

2. 入门指南

2.1 系统要求

CloudDrive Modern 可在任何现代网络浏览器中运行。为获得最佳性能，我们建议：

  • 浏览器： Chrome 90+、Firefox 88+、Safari 14+ 或 Edge 90+
  • 网络连接： 稳定的宽带连接用于云操作
  • 屏幕分辨率： 最低 1024x768（响应式设计适应所有尺寸）
  • JavaScript： 必须启用

2.2 安装说明

CloudDrive Modern 是一个无需安装的网络应用程序。只需：

1 打开您的网络浏览器，导航到管理员提供的 CloudDrive Modern URL。
2 将页面添加到书签以便将来轻松访问。
3 对于移动设备，您可以将网络应用添加到主屏幕以获得类似原生应用的体验。

2.3 首次登录

1 在登录屏幕上输入您的用户名和密码。
2 点击"登录"按钮或按 Enter。
3 如果您没有账户，点击"注册"创建新账户。
💡 提示： 启用"记住我"以在浏览器会话之间保持登录状态。出于安全考虑，在公共计算机上此功能被禁用。

3. 用户界面

3.1 导航

CloudDrive Modern 界面由几个关键区域组成：

  • 侧边栏： 文件夹树导航和驱动器快速访问
  • 标题栏： 视图控件、排序选项和操作按钮
  • 主内容区： 文件和文件夹显示区域
  • 属性面板： 详细文件信息（可切换）
  • 上下文菜单： 右键点击快速操作

3.2 查看模式

CloudDrive Modern 提供三种查看模式以适应不同的偏好：

查看模式 描述 最适合
网格视图 文件显示为带缩略图的卡片 可视化浏览、图像集合
列表视图 带文件详细信息的紧凑表格 详细信息、大型文件列表
瀑布视图 Pinterest 风格的砖石布局 混合媒体、可视化探索
💡 提示： 使用键盘快捷键 V 可以快速切换查看模式。

3.3 排序和过滤

使用强大的排序和过滤选项高效组织您的文件：

排序选项

  • 名称： 字母顺序（A-Z 或 Z-A）
  • 大小： 文件大小（从小到大或反之）
  • 类型： 文件扩展名/类型
  • 修改日期： 最后修改时间

高级过滤

  • 按文件名或内容搜索
  • 按文件类型过滤（图像、视频、文档等）
  • 日期范围过滤
  • 大小范围过滤

4. 文件操作

4.1 上传文件

CloudDrive Modern 支持多种上传方法：

方法1：拖放

1 从计算机的文件资源管理器中选择文件或文件夹。
2 将它们拖到 CloudDrive 窗口上。
3 将它们放在所需位置（主区域或特定文件夹）。

方法2：上传按钮

1 点击标题栏中的"上传"按钮。
2 从文件选择对话框中选择文件。
3 点击"打开"开始上传。
💡 提示： 上传进度通过进度条和通知实时显示。大文件分块上传以确保可靠性。

4.2 下载文件

单独或批量下载文件：

单文件下载

  • 右键点击文件并选择"下载"
  • 双击非媒体文件
  • 使用属性面板中的下载按钮

多文件下载

1 使用 Ctrl+点击 或 Shift+点击 选择多个文件。
2 点击标题栏中的"下载"按钮。
3 文件将被打包并作为 ZIP 压缩包下载。

4.3 移动和复制

通过在文件夹间移动或复制来组织文件：

拖放方法

1 选择要移动的文件。
2 将它们拖到侧边栏或主区域中的目标文件夹。
3 放置时按住 Ctrl 进行复制而非移动。

上下文菜单方法

1 选择文件并右键点击打开上下文菜单。
2 选择"移动到..."或"复制到..."。
3 在文件夹选择对话框中选择目标文件夹。
⚠️ 警告： 在不同云存储提供商之间移动文件可能需要更长时间，因为需要传输数据。操作将在后台执行。

4.4 重命名和删除

重命名文件和文件夹

  • 右键点击并选择"重命名"
  • 选择文件时按 F2
  • 点击属性面板中的编辑按钮

删除文件和文件夹

  • 选择项目并按 Delete
  • 右键点击并选择"删除"
  • 使用标题栏中的删除按钮
⚠️ 警告： 删除的文件可能无法恢复，具体取决于云存储提供商的政策。删除重要文件前请务必确认。

5. 文件夹管理

5.1 创建文件夹

1 点击标题栏中的"+"按钮或在空白区域右键点击。
2 从菜单中选择"新建文件夹"。
3 输入文件夹名称并按 Enter 或点击"创建"。

5.2 加密文件夹

为敏感数据创建加密文件夹：

1 点击"+"按钮并选择"新建加密文件夹"。
2 输入文件夹名称和强密码。
3 确认密码并点击"创建"。
⚠️ 重要： 加密文件夹密码无法恢复。请确保记住密码或安全存储。

5.3 文件夹导航

高效导航文件夹结构：

  • 侧边栏树： 点击树视图中的文件夹进行快速导航
  • 面包屑导航： 点击路径的任何部分跳转到该级别
  • 前进/后退： 使用浏览器前进/后退按钮或键盘快捷键
  • 向上按钮： 点击转到父文件夹

6. 云存储集成

6.1 添加云存储

连接您的云存储账户在一个地方管理所有文件：

1 点击"+"按钮并选择"添加云存储"。
2 选择您的云存储提供商（Google Drive、OneDrive 等）。
3 按照身份验证流程授予访问权限。
4 您的云存储将出现在侧边栏中。

6.2 管理云账户

  • 刷新： 右键点击云存储并选择"刷新"进行同步
  • 断开连接： 移除对云存储账户的访问
  • 配额： 查看存储使用情况和可用空间
  • 设置： 配置同步选项和首选项

7. 高级功能

7.1 媒体查看器

CloudDrive Modern 包含强大的内置媒体查看器：

支持的格式

  • 图像： JPEG、PNG、GIF、WebP、SVG、BMP
  • 视频： MP4、WebM、AVI、MOV（取决于浏览器）
  • 音频： MP3、WAV、OGG、AAC
  • 文档： PDF 预览

媒体查看器控件

  • 导航： 箭头键或屏幕按钮
  • 缩放： 鼠标滚轮或捏合手势
  • 全屏： 按 F 或点击全屏按钮
  • 幻灯片： 自动播放媒体文件

7.2 外部播放器

使用外部应用程序启动文件以增强播放：

  • VLC Media Player： 用于高级视频播放
  • PotPlayer： Windows 媒体播放器集成
  • IINA： macOS 媒体播放器
  • Infuse： iOS/tvOS 媒体播放器
  • nPlayer： 跨平台播放器
💡 提示： 外部播放器需要安装在您的设备上并配置为处理 CloudDrive URL。

7.3 搜索功能

使用强大的搜索功能快速查找文件：

基本搜索

1 点击搜索图标或按 Ctrl+F。
2 输入搜索查询。
3 按 Enter 进行搜索。

高级搜索选项

  • 模糊匹配： 查找具有相似名称的文件
  • 内容搜索： 在文件内容中搜索（在支持的情况下）
  • 文件类型过滤： 将搜索限制为特定文件类型
  • 日期范围： 查找在特定时间段内修改的文件

7.4 文件共享

与他人共享文件和文件夹：

1 右键点击文件或文件夹并选择"共享"。
2 配置共享权限和过期时间。
3 复制生成的共享链接。
4 将链接发送给接收者。

7.5 离线下载

排队下载以进行离线访问：

  • 将文件添加到离线下载队列
  • 下载在后台继续
  • 在没有网络时访问文件
  • 管理下载优先级和带宽

9. 故障排除

常见问题及解决方案

上传失败

  • 大文件： 检查云存储的文件大小限制
  • 网络问题： 验证网络连接稳定性
  • 存储已满： 检查可用存储空间
  • 文件类型： 某些云提供商限制特定文件类型

登录问题

  • 错误凭据： 验证用户名和密码
  • 账户锁定： 联系您的管理员
  • 浏览器问题： 清除缓存和 Cookie
  • 网络： 检查防火墙和代理设置

性能问题

  • 加载缓慢： 尝试减少每页显示的文件数量
  • 内存使用： 关闭未使用的浏览器标签
  • 网络速度： 检查您的网络连接
  • 浏览器缓存： 清除浏览器缓存并刷新

云存储问题

  • 身份验证过期： 重新认证您的云账户
  • 同步冲突： 手动解决文件冲突
  • API 限制： 等待速率限制重置
  • 服务中断： 检查云提供商状态页面

10. 常见问题

问：我在 CloudDrive Modern 中的数据安全吗？

答：是的，CloudDrive Modern 使用行业标准的安全实践，包括 HTTPS 加密、安全身份验证和可选文件加密。您的云存储凭据经过加密并安全存储。

问：我可以离线访问 CloudDrive Modern 吗？

答：CloudDrive Modern 的大部分功能需要网络连接。但是，您可以使用离线下载功能将文件缓存到本地以进行离线访问。

问：我有多少存储空间？

答：存储空间取决于您的云存储账户和本地服务器配置。在侧边栏或属性面板中查看存储信息。

问：我可以与没有账户的人共享文件吗？

答：是的，您可以创建共享链接，允许外部用户访问特定文件或文件夹，而无需账户。

问：如果我删除文件会发生什么？

答：删除的文件将从 CloudDrive Modern 中移除，并可能根据提供商的政策移动到云存储提供商的回收站中。

问：我可以上传文件夹吗？

答：是的，您可以拖放整个文件夹，CloudDrive Modern 将在上传过程中保留文件夹结构。

问：有移动应用吗？

答：CloudDrive Modern 是一个在移动设备上运行的响应式网络应用程序。您可以将其添加到设备的主屏幕以获得类似原生应用的体验。

问：如何报告错误或请求功能？

答：联系您的系统管理员或使用应用程序设置菜单中的反馈选项。

💡 需要更多帮助？ 联系您的系统管理员或 IT 支持团队以获得 CloudDrive Modern 的额外帮助。
